# SplitTable — Legal

Public legal docs for the SplitTable iOS app, served via GitHub Pages.

- `privacy.html` — Privacy Policy (GDPR-aligned)
- `terms.html` — Terms of Service
- `index.html` — landing page linking both

URLs once Pages is enabled:

- `https://mourad149.github.io/splittable-legal/`
- `https://mourad149.github.io/splittable-legal/privacy.html`
- `https://mourad149.github.io/splittable-legal/terms.html`

Update the iOS `PrivacySafetyView.privacyURL` / `termsURL` constants
once these resolve.

**Both documents are placeholder copy** suitable for staging /
TestFlight. They cover the GDPR essentials and Apple's App Store
privacy requirements but should be reviewed by a French lawyer
before production launch.
